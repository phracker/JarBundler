Listing of Java and Bundle Properties Jar Bundler Sets

This section lists the Java and bundle properties that Jar Bundler can set. They are listed according to each pane in Jar Bundler.

Build Information pane:

  • Main Class = Java/MainClass

  • Arguments to Main = Java/Arguments

  • JVM Version = Java/JVMVersion

  • Custom Icon = CFBundleIconFile


Checkbox 1.4 & 1.5 Property
Use MacOS Menu Bar apple.laf.useScreenMenuBar
Anti-alias Text apple.awt.textantialiasing
Anti-alias Graphics apple.awt.antialiasing

The Classpath and Files pane sets only one bundle property. The classpath listing is equated to the Java/ClassPath bundle property.

Properties pane:

  • Type = CFBundlePackageType

  • Signature = CFBundleSignature

  • Version = CFBundleVersion

  • Identifier = CFBundleIdentifier

  • Heap Minimum Size adds the value to Java/VMOptions

  • Heap Maximum Size adds the value to Java/VMOptions

  • Get-Info String = CFBundleGetInfoString

  • Short Version = CFBundleShortVersionString

  • VM Options = Java/VMOptions

  • Allow Mixed Localizations = CFBundleAllowMixedLocalizations

  • Development Region = CFBundleDevelopmentRegion

  • Bundle Name = CFBundleName

  • Info Dictionary Version = CFBundleInfoDictionaryVersion

  • About Menu Name = com.apple.mrj.application.apple.menu.about.name

  • Set Working directory to inside Application package sets Java/WorkingDirectory to $APP_PACKAGE/Contents/Resources/Java
